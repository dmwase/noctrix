# Noctrix — AI-Native Threat Operations Platform

Investor/customer pitch demo dashboard — fully static, no backend required.

**Live demo:** https://YOUR-USERNAME.github.io/noctrix

## Deploy to GitHub Pages (3 steps)

1. Create a new GitHub repo named `noctrix`
2. Upload all 4 files to the repo root:
   - `index.html`
   - `404.html`
   - `.nojekyll` ← important: note the dot prefix
   - `README.md`
3. Go to **Settings > Pages > Deploy from branch > main > / (root) > Save**

Live at `https://YOUR-USERNAME.github.io/noctrix` within ~60 seconds.
